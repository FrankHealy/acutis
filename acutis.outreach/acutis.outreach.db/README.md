# Acutis Outreach database boundary

This directory records database ownership for Acutis Outreach.

Outreach currently has no database project, `DbContext`, schema, migrations, connection-string key, or database deployment. None was created as part of the repository reorganisation.

Any future Outreach persistence requires a separately reviewed design and initial migration. It must not reuse or couple to the Centre, Community, Practitioner, or legacy Ambulatory databases.
