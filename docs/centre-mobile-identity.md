# Centre mobile identity continuity

The existing Centre tablet application remains in `acutis.tab` with scheme `acutis-tab`, Android package `com.salientrecovery.acutis.tab`, and its existing EAS project ID. These legacy identifiers are intentionally retained to preserve installed-build upgrades, secure token storage, signing, and update continuity. New product applications use the approved product-specific schemes and reverse-domain identifiers.
