# Centre mobile identity continuity

The existing Centre tablet application is physically located at `acutis.centre/acutis.centre.tab`. Its package name `acutis.tab`, scheme `acutis-tab`, Android package `com.salientrecovery.acutis.tab`, and existing EAS project ID are intentionally retained to preserve installed-build upgrades, secure token storage, signing, and update continuity. New product applications use the approved product-specific schemes and reverse-domain identifiers.
